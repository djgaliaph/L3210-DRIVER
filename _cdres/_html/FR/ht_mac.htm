Ajoutez l'imprimante que vous configurez. Les étapes suivantes sont basées sur Mac OS X 10.8. En fonction de la version du système d'exploitation, le menu et les étapes peuvent différer.
Si votre imprimante apparaît déjà en tant qu'« EPSON XX-XXXX AirPrint », sélectionnez l'imprimante, puis cliquez sur [-] pour la supprimer.
Cliquez sur [+], puis sélectionnez [Ajouter une imprimante ou un scanner].
Ne sélectionnez pas une imprimante affichée sur ce menu. En sélectionnant le menu d'ajout d'imprimante, vous pouvez sélectionner le pilote d'impression Epson.
Sélectionnez l'imprimante que vous souhaitez enregistrer depuis la liste.
Confirmez le type de connexion de l'imprimante depuis [Type], puis sélectionnez l'imprimante.
Cliquez sur [Utiliser], puis sélectionnez « EPSON XX-XXXX ».
Pour utiliser toutes les fonctions de l'imprimante, sélectionnez le pilote d'impression Epson. Si vous sélectionnez « Secure AirPrint » ou « AirPrint », vous ne pouvez utiliser que les fonctions de base de l'imprimante.
Cliquez sur [Ajouter].
Confirmez que l'imprimante est enregistrée, puis fermez l'écran.
