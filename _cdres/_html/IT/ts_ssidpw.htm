Selezionare l'SSID della banda a 2,4 GHz del router e immettere la password di seguito.

Individuare SSID e password della banda a 2,4 GHz sull'etichetta o sulla scheda del router oppure sul relativo manuale. La password può essere una chiave.
Nota: l'SSID potrebbe includere ""g"", ""G"" o ""b/g/n"". La password può essere una chiave di rete o Wi-Fi oppure una password wireless. Lo stile varia in base al produttore del router.
