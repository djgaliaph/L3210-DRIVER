Troubleshooting

Cannot find the product

Make sure the product and the access point (or wireless router) are turned on.
If you use the product in Ethernet network, make sure the product is connected to the hub with a LAN cable correctly.
If access restrictions are set on the access point (or wireless router), register the product's MAC address or IP address to the access point. Check the MAC address on the product's control panel or network status sheet.
If a privacy separator feature of the access point (or wireless router) is enable, disable it. For more details, see the manual of the access point (or wireless router).
Check the IP address is correct. It should eliminate overlap with other ones and you cannot connect the product to a different network segment.

Cannot connect the product to the network

Check the communication is established between the computer and the access point (or wireless router).
Check the SSID (Network name) used on the access point is selected or entered correctly. You cannot use non-ASCII characters.
The password is case-sensitive. Check the password is entered correctly.
Move the product closer to the access point or adjust the access point's external antenna.
Radio wave interference from other wireless devices may interrupt wireless connection. Check that other wireless devices nearby are not turned on.
