故障排除

未找到产品

确保产品和接入点（或无线路由器）已打开。
如果在以太网中使用该产品，请确保已将产品正确连接到有局域网线缆的集线器上。
如果接入点（或无线路由器）上设置了接入限制，则请在接入点上注册产品的 MAC 地址或 IP 地址。检查产品的控制面板或网络状态页上的 MAC 地址。
如果启用了接入点（无线路由器）的隐私分离器功能，则将其禁用。有关更多详情，请参阅接入点（无线路由器）手册。
检查 IP 地址是否正确。应消除与其他人重叠的 IP 地址，并且不能将产品连接至另一不同网段。

无法将产品连接到网络

检查计算机和接入点（无线路由器）之间的通信是否建立。
检查接入点上使用的 SSID（网络名称）是否选定，输入是否正确。不可使用非 ASCII 字符。
密码区分大小写。检查输入的密码是否正确。
将产品靠近接入点，并调整接入点的外部天线。
其他无线设备的无线电波干扰可能会中断无线连接。检查附近的其他无线设备是否为未打开状态。
