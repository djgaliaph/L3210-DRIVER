Lägg till skrivaren som ställs in. Följande steg är baserade på Mac OS X 10.8. Beroende på operativsystemets version kan menyn och stegen att utföras variera.
Om din skrivare redan har visats som "EPSON XX-XXXX AirPrint", välj skrivaren och klicka sedan på [-] för att ta bort.
Klicka på [+] och välj sedan [Lägg till skrivare eller skanner].
Välj inte en skrivare som visas i den här menyn. Genom att välja menyn för att lägga till skrivare kan du välja Epson-skrivardrivrutinen.
Välj den skrivare som du vill registrera i listan.
Bekräfta skrivarens anslutningstyp i [Typ] och välj sedan skrivaren.
Klicka på [Använd] och välj sedan "EPSON XX-XXXX".
För att kunna använda alla skrivarfunktioner, välj Epson skrivardrivrutin. Om du väljer "Säker AirPrint" eller "AirPrint" kan du endast använda skrivarens grundläggande funktioner.
Klicka på [Lägg till].
Bekräfta att skrivaren är registrerad och stäng sedan skärmen.
