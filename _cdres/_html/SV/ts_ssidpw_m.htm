Välj SSID för 2,4 GHz-bandet på din routers nedan och ange lösenordet vid uppmaning.

Leta upp SSID och lösenord för 2,4 GHz-bandet på routerns etikett eller kort, eller i dess bruksanvisning. Lösenordet kan kallas för en nyckel.
Obs! SSID kan inkludera ett "g", "G" eller "b/g/n". Lösenordet kan vara ett nätverk eller Wi-Fi-nyckel, eller ett trådlöst lösenord. Det varierar beroende på routerns tillverkare.
