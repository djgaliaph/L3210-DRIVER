疑難排解

找不到產品

請確實開啟產品和無線網路基地台 (或無線路由器) 的電源。
如果在乙太網路內使用本產品，必須使用 LAN 連接線將產品正確連接至集線器。
如果無線網路基地台 (或無線路由器) 設有存取限制，請在無線網路基地台內登錄產品的 MAC 位址或 IP 位址。可從產品的控制面板或網路狀態表查看 MAC 位址。
如果無線網路基地台 (或無線路由器) 啟用了私密分隔區功能，請停用該功能。詳細資料請參閱無線網路基地台 (或無線路由器) 的手冊。
查看 IP 位址是否正確。產品的 IP 不應與其他裝置的 IP 重複，也不可將產品連接至其他的網路區段。

無法將產品連接至網路

查看電腦和無線網路基地台 (或無線路由器) 之間是否建立通訊。
查看無線網路基地台上使用的 SSID (網路名稱) 是否正確選取或輸入。不可使用非 ASCII 字元。
密碼需區分大小寫。查看密碼是否正確輸入。
移動產品，減少與無線網路基地台之間的距離，或調整無線網路基地台的外接天線。
其他無線裝置的無線電波干擾，可能使無線連線中斷。查看附近的其他無線裝置是否開啟電源。
