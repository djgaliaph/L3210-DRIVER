문제 해결

제품을 찾을 수 없습니다.

제품과 액세스 포인트(또는 무선 라우터)가 켜져 있는지 확인합니다.
이더넷 네트워크에서 제품을 사용하는 경우 제품이 LAN 케이블을 통해 허브에 올바르게 연결되었는지 확인합니다.
액세스 제한 사항이 액세스 포인트(또는 무선 라우터)에 설정된 경우 액세스 포인트에 제품의 MAC 주소 또는 IP 주소를 등록합니다. 제품의 제어판이나 네트워크 상태 시트에서 MAC 주소를 확인합니다.
액세스 포인트(또는 무선 라우터)의 프라이버시 구분자(Privacy Separator) 기능이 사용되는 경우 해당 기능을 사용하지 않도록 설정합니다. 자세한 내용은 액세스 포인트(또는 무선 라우터) 설명서를 참조하십시오.
IP 주소가 올바른지 확인합니다. 겹치는 다른 IP 주소를 제거해야 합니다. 그렇지 않으면 제품을 다른 네트워크 세그먼트에 연결할 수 없습니다.

제품을 네트워크에 연결할 수 없습니다.

컴퓨터와 액세스 포인트(또는 무선 라우터) 간의 통신이 설정되었는지 확인합니다.
액세스 포인트에서 사용되는 SSID(네트워크 이름)가 올바르게 선택 또는 입력되었는지 확인합니다. ASCII 문자가 아닌 문자는 사용할 수 없습니다.
비밀번호는 대/소문자를 구분합니다. 비밀번호가 올바르게 입력되었는지 확인합니다.
제품을 액세스 포인트 가까이로 이동하거나 액세스 포인트의 외부 안테나를 조정합니다.
다른 무선 장치의 전파 간섭이 무선 연결을 방해할 수 있습니다. 부근의 다른 무선 장치가 켜져 있지 않은지 확인합니다.
